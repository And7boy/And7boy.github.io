1.2.1
编程时注意定义的变量类型
例如
void fun(int a, int b, long *c)
{
int a1,a2,b1,b2;
a1=a%10;
a2=(a/10)%10;
b1=b%10;
b2=(b/10)%10;

*c=a1*1000+a2*10+b2+b1*100;

}
其中勿将*c赋值变成了c赋值成地址了。

2.3.1
编程时的自定义函数如果要求返回要用return
变换类型写法必须严格，例如（int）x
四舍五入灵活写法：
double fun(double x)
{
x=(int)(x100+0.5); /*移动小数点/
x/=100; /舍掉小数点后两位/
return x;
}
加0.5完事
