文件搜索命令

文件搜索命令find

注意：该命令占用资源过多，谨慎使用
命令名称：find
所在路径：/bin/find
执行权限：所有用户
语法：find[搜索范围][匹配条件]
功能描述：文件搜索
使用例

以及-a的活用

locate 在文件资料库中查找文件

命令名称：locate
所在路径：/bin/locate
执行权限：所有用户
语法：locate 文件名
功能描述：在文件资料库中查找文件
注意
updatedb可以更新文件资料库
tmp文件夹中是搜不到的
locate -i 不区分大小写

which搜索命令所在目录及别名信息

命令名称：which
所在路径：/usr/bin/which
执行权限：所有用户
语法：which 命令
功能描述：搜索命令所在目录及别名信息
注意
usr下的命令只有管理员能用

grep在文件中搜寻字串匹配的行并输出

命令名称：grep
所在路径：/bin/grep
执行权限：所有用户
语法：grep -iv[指定字串][文件]
功能描述：在文件中搜寻字串匹配的行并输出
-i 不区分大小写
-v 排除指定字串
注意
-v ^# 去除行首注释
